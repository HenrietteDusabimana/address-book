# address-book
Web app address book
 
 
This is a web app for address book. 
It is built using Django. In case you want to run the project these are the steps to follow:

- Make sure that you have the latest version of python or at least version >=3.0 installed.
- Install Django using this command:
   * Windows: #py -m pip install Django
   * Others:  #python -m pip install Django
For more details about Django follow this link: https://docs.djangoproject.com/en/3.2/topics/install/

- In the project folder install some dependencies that were used in development by using this command:
  pip install -r requirements.txt
  
- To start the server run this command:
  #python manage.py runserver
  
  You should be able to access the project on this link
   http://127.0.0.1:8000/
